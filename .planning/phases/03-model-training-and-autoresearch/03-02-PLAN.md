3. **load_and_split(df: pd.DataFrame) -> tuple:**
- Define feature_cols = [c for c in df.columns if c not in META_COLS + [TARGET]]
- Assert no FORBIDDEN_FEATURES in feature_cols
- Temporal split: train = df[df["season"].between(2005, 2022)], val_2023 = df[df["season"] == 2023], val_2022 = df[df["season"] == 2022], val_2021 = df[df["season"] == 2021]
- Drop ties: .dropna(subset=[TARGET]) on all splits
- Drop NaN rolling rows: .dropna(subset=feature_cols) on all splits
- NOTE: 2021 and 2022 remain IN the training set (train on ALL of 2005-2022). The val_2022/val_2021 subsets are for in-sample evaluation on the already-trained model, NOT held-out validation. These are in-sample training accuracies useful as overfitting signals (if in-sample accuracy drops while out-of-sample 2023 improves, something is wrong). Only 2023 is true held-out validation. Add a docstring comment in load_and_split making this distinction explicit.
- Print row counts: f"Train: {len(train)}, Val 2023 (held-out): {len(val_2023)}, Eval 2022 (in-sample): {len(val_2022)}, Eval 2021 (in-sample): {len(val_2021)}"
- Return (train, val_2023, val_2022, val_2021, feature_cols)

4. **train_and_evaluate(X_train, y_train, X_val_2023, y_val_2023, X_val_2022, y_val_2022, X_val_2021, y_val_2021, params: dict) -> tuple[dict, XGBClassifier]:**
- Create XGBClassifier with objective="binary:logistic", eval_metric="logloss", use_label_encoder=False, **params
- Fit with eval_set=[(X_val_2023, y_val_2023)], verbose=False
- Compute: accuracy_score, log_loss, brier_score_loss for 2023 using predict and predict_proba
- Compute: accuracy_score for 2022 and 2021 using predict
- Compute TreeSHAP: shap.TreeExplainer(model).shap_values(X_val_2023), then np.abs(shap_values).mean(axis=0), sort descending, take top 5 as list of (feature_name, importance_float) tuples
- Return (results_dict, model) where results_dict has keys: val_accuracy_2023, val_accuracy_2022, val_accuracy_2021, log_loss, brier_score, shap_top5

5. **log_experiment(experiment_id, params, features_used, val_acc_2023, val_acc_2022, val_acc_2021, baseline_home, baseline_record, log_loss_val, brier_score_val, shap_top5, keep, hypothesis, prev_best_acc, model_path, jsonl_path="models/experiments.jsonl"):**
- Accept optional `jsonl_path` parameter (default "models/experiments.jsonl") for testability — tests pass tmp_path to avoid writing to real project files
- Build entry dict with all fields matching the locked schema from CONTEXT.md:
  experiment_id, timestamp (UTC ISO), params, features, val_accuracy_2023, val_accuracy_2022, val_accuracy_2021, baseline_always_home, baseline_better_record, log_loss, brier_score, shap_top5 (as list of {feature, importance} dicts), keep, hypothesis, prev_best_acc, model_path
- Append JSON line to jsonl_path (mode="a")
- Log to MLflow using context manager: mlflow.start_run(run_name=f"exp-{experiment_id:03d}")
  - mlflow.log_params(params)
  - mlflow.log_metrics with all numeric values
  - mlflow.set_tag("keep", str(keep))
  - mlflow.set_tag("hypothesis", hypothesis)
  - If model_path: mlflow.log_artifact(model_path)

6. **save_model(model, experiment_id, artifacts_dir="models/artifacts") -> str:**
- os.makedirs(artifacts_dir, exist_ok=True)
- path = os.path.join(artifacts_dir, f"model_exp{experiment_id:03d}.json")
- model.save_model(path)
- Return path

7. **save_best_model(model, artifacts_dir="models/artifacts") -> str:**
- path = os.path.join(artifacts_dir, "best_model.json")
- model.save_model(path)
- Return path

8. **should_keep(new_acc, prev_best_acc, new_log_loss, prev_best_log_loss) -> bool:**
- acc_improvement = new_acc - prev_best_acc
- If acc_improvement >= 0.005: return True (>=0.5 percentage points)
- If acc_improvement > 0 and new_log_loss < prev_best_log_loss: return True
- Otherwise: return False

Module imports: pandas, numpy, json, os, datetime (timezone, datetime), xgboost (XGBClassifier), sklearn.metrics (accuracy_score, log_loss, brier_score_loss), shap, mlflow, features.definitions (TARGET, FORBIDDEN_FEATURES).

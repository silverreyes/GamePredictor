3. For each season:
   a. **PBP ingestion:**

      - Select curated columns: `select_pbp_columns(df)` -- raises KeyError on schema drift (hard fail)

      - Upsert to raw_pbp table using SQLAlchemy PostgreSQL `insert(...).on_conflict_do_update(index_elements=["game_id", "play_id"], set_=update_cols)` -- process in chunks of 5000 rows to avoid memory issues

      - Log to ingestion_log table

   b. **Schedule ingestion:**
      - Load via `load_schedules_cached(season)`
      - Filter: keep rows where game_type != 'PRE' (exclude preseason, keep REG + playoff types like WC/DIV/CON/SB)
      - Select curated columns: `select_schedule_columns(df)`
      - Normalize teams: `normalize_teams_in_df(df, TEAM_COLUMNS_SCHEDULE)`
      - Add `games_in_season` column: 16 if season <= 2020 else 17
      - Upsert to schedules table using `insert(...).on_conflict_do_update(index_elements=["game_id"], set_=update_cols)`
      - Count regular season games: count rows WHERE game_type == 'REG'
      - Log to ingestion_log table

   c. **Validation:**
      - Call `validate_game_count(season, reg_game_count_pbp, "raw_pbp")` (PBP validation counts distinct game_id from regular season plays)
      - Call `validate_game_count(season, reg_game_count_schedule, "schedules")`
      - Collect all ValidationResult objects

4. After ALL seasons processed: call `print_validation_summary(results)`
5. If any validation failed: `sys.exit(1)` (per CONTEXT.md: finish all, then fail)

**Upsert helper** (in data/ingest.py or data/db.py):
```python
from sqlalchemy.dialects.postgresql import insert as pg_insert

def upsert_dataframe(engine, table, df, conflict_columns, chunk_size=5000):
    """Upsert DataFrame rows into PostgreSQL table in chunks."""
    records = df.to_dict(orient="records")
    for i in range(0, len(records), chunk_size):
        chunk = records[i:i + chunk_size]
        stmt = pg_insert(table).values(chunk)
        update_cols = {
            c.name: stmt.excluded[c.name]
            for c in table.columns
            if c.name not in conflict_columns
        }
        upsert_stmt = stmt.on_conflict_do_update(
            index_elements=conflict_columns,
            set_=update_cols,
        )
        with engine.begin() as conn:
            conn.execute(upsert_stmt)
```

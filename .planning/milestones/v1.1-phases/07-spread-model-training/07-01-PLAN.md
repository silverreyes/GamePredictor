**In models/train_spread.py, fix train_and_evaluate_spread() lines 195-199:**

Change from:
```python
model = XGBRegressor(
    objective="reg:squarederror",
    eval_metric="mae",
    **params,
)
```

To:
```python
objective = params.pop("objective", "reg:squarederror")
model = XGBRegressor(
    objective=objective,
    eval_metric="mae",
    **params,
)
```

This extracts `objective` from the params dict with a default of `"reg:squarederror"`, preventing TypeError when `objective` is passed in `EXPERIMENT_PARAMS`. The `params.pop()` mutates the dict, which is fine since params is a copy made from `EXPERIMENT_PARAMS` at the call site (`run_spread_experiment` passes `EXPERIMENT_PARAMS` which is a dict literal copy via `{**DEFAULT_SPREAD_PARAMS, ...}`).

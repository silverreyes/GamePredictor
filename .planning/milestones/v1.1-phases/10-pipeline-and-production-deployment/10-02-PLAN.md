    1. Load spread model and experiment info:

       ```python
       model = load_best_spread_model()
       ```

    5. Get schedule data for game dates and actual scores:
       ```python
       engine = get_engine()
       schedule_query = """
           SELECT game_id, gameday, home_score, away_score
           FROM schedules
           WHERE season = %(season)s AND game_type = 'REG'

       schedule_info = pd.read_sql(schedule_query, engine, params={"season": season})
       schedule_map = {row["game_id"]: row for _, row in schedule_info.iterrows()}
       ```

    6. Build prediction records with BOTH predicted and actual columns:
       For each completed game:
       - `predicted_spread = float(predicted_spreads[i])`
       - `predicted_winner = home_team if predicted_spread >= 0 else away_team` (home-team convention for zero per Phase 8 decision)
       - `actual_spread = float(sched["home_score"] - sched["away_score"])` (home_score minus away_score, matching prediction convention)
       - `actual_winner = home_team if actual_spread > 0 else away_team` (for actual_spread == 0 i.e. ties, set actual_winner to home_team)
       - `correct = (predicted_winner == actual_winner)`
       - Record dict keys: game_id, season, week, game_date, home_team, away_team, predicted_spread, predicted_winner, model_id, actual_spread, actual_winner, correct

    7. Print summary: total predictions, count correct, accuracy %.
